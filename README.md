# RCIMDemo
第三方融云的一个Demo

![image](https://github.com/zhengwenming/RCIM/raw/master/RCIM/Resoures/images/RCIM.gif) 

本人用过融云的1.4.4和2.2.4版本的SDK，分别用于对两款APP的即时通讯功能开发集成，主要用了IMKit类，功能为单聊（聊天室和群聊不涉及，理解了单聊，聊天室和群聊也就不攻而破）,吐槽一下关于iOS端融云SDK集成资料很少，只能看融云官方的开发文档，由于开发文档的语言和关键词用的也比较官方比较专业化，我想要通俗易通，雅俗共赏的资料，没有！没有！真的没有！博客之类的资料只能找到安卓的文章，无奈只能iOS小屌丝看安卓大神写的关于融云集成的android代码，小伙伴们看的是很蛋疼。现在福利来了，iOS端的融云集成会不断更新，持续关注。想集成融云聊天功能的小伙伴不防先普及下理论知识，防止以后走弯路又或者总是问些不靠谱的问题，“我为什么不能聊天”，“为什么头像和名字出不来”等等一系列问题，文章中如果有错误的地方希望大家提出，本人会及时改正，避免误人子弟，高手勿喷，哈哈。。。
